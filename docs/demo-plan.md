# TRADESIGHT NSW Demonstration Plan

## Purpose

Demonstrate TRADESIGHT as a NSW-focused construction compliance and project intelligence workflow.

The demonstration is not a certified compliance decision. It shows disciplined intake, evidence handling, agent routing, assessment scoring, and audience-specific reporting.

## Demonstration audiences

- Homeowner
- Builder or trade professional
- Government or authority reviewer

## Demonstration sequence

1. Select a demo case.
2. Enter or load the intake text.
3. Add the supplied evidence note.
4. Show the runtime events panel.
5. Show assessment state, priority, evidence completeness, and rule provenance.
6. Show agent orchestration.
7. Show Report Studio output.
8. Show project record status.
9. Explain unresolved items and boundaries.

## Demo cases

Demo cases are stored in `src/demo/demo-cases.js`.

## Presentation mode

Presentation mode is stored in `src/demo/presentation-mode.js` and runs the demo cases through the same TRADESIGHT runtime used by the application.

## Core message

TRADESIGHT does not replace qualified review. It turns scattered construction problems into structured evidence, conservative assessment pathways, agent-routed workflows, and audience-ready reports.
